Wordpress Plugins

A list of plugins that I regularly use for Wordpress sites.

  • ACF, create custom fields for a specific page or template. Use case examples: making text editable, images for a carousel or just any custom content field really.

  • Updraft, setup automatic backups that can be automatically uploaded to a remote cloud storage.

  • Polylang, simple site translation (pages, strings and medias).

  • Contact Form 7, basic form plugin with easy wp_mail integration.

  • Akismet, filter spam comments, free for personal use.

  • EWWW Image Optimizer, lossless image optimization on upload.

  • W3 Total Cache, caching and code minification.

  • Cookie Notice, GDPR cookie banner.
